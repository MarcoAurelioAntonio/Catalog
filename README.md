## Prerequisites
Have `Ruby`, and `Git` installed on your local machine.

### Setup

Clone this repository to your desired folder:

- git clone https://github.com/MarcoAurelioAntonio/Catalog
- cd Catalog
- Run `bundle install` to install the required gems
